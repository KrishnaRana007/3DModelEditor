# 3D Model Editor v27

## Camera Objects

Added camera objects and camera-view workflow.

### Added
- Left toolbar Camera flyout.
- Perspective Camera object.
- Orthographic Camera object.
- View menu camera actions:
  - View Through Selected Camera
  - Exit Camera View
- Context menu camera actions.
- Viewport camera preview frame overlay while viewing through a camera.
- Scene hierarchy camera icons.
- Camera inspector controls:
  - View Through Camera
  - Exit Camera View
  - Near
  - Far
  - Zoom
  - FOV for Perspective Camera
  - Ortho Size for Orthographic Camera
- Camera save/load support in custom JSON/XML project files.
- Camera transform animation support through the existing Animation tab.
- Camera projection property animation support for FOV, Zoom, Near, Far, and Ortho Size.

### Compatibility
- Cameras can be selected, moved, rotated, scaled, renamed, hidden, locked, duplicated, deleted, parented, and saved/loaded.
- Existing tools, hierarchy, animation, material, import/export, and project save/load features are preserved.
