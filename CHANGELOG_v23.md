# 3D Model Editor v23

## Animation Dope Sheet

- Added Dope Sheet rows inside the Animation tab.
- Shows selected-object keyframes as separate rows for:
  - Position
  - Rotation
  - Scale
  - Material Color
  - Opacity
- Keyframe diamonds can be clicked to jump to their frame.
- Keyframe diamonds can be dragged horizontally to move keys to another frame.
- Keyframe diamonds can be deleted by double-clicking or right-clicking.
- Dope Sheet updates when selection changes and remains compatible with existing Play/Pause, Spacebar playback, timeline scrubbing, Auto Key, Add Key, and multi-object playback.
