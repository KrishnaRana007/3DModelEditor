# Changelog v12 - Left Toolbar Alignment Fix

## Fixed
- Object and Light buttons in the left toolbar now use the same full-width alignment as Select, Translate, Scale, and Rotate.
- Object and Light buttons no longer shrink to content width.
- Flyout behavior remains unchanged: Object and Light options still open to the right side in multi-row icon grids.

## Preserved
- Offline/local Three.js setup.
- Default cube at origin.
- Bottom Inspector / Material Asset / Properties tabs.
- Object / Face / Edge / Vertex modes.
- Context menu and submenu behavior.
- Undo / Redo command pattern.
- Modeling tools and guide modal.
