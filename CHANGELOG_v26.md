# 3D Model Editor v26

## Scene Hierarchy Improvements

Added Project Structure tools:

- Rename object from the hierarchy row.
- Hide/show object using the eye button.
- Lock/unlock object using the lock button.
- Duplicate object from the hierarchy row.
- Delete object from the hierarchy row.
- Parent/child hierarchy using drag-and-drop.
- Drag an object row onto another object row to parent it.
- Drag an object row onto the Scene Root drop area to unparent it.
- Hierarchy relationships, visibility, lock state, and parent IDs are included in custom project JSON/XML save/load.

Locked objects cannot be selected or transformed until unlocked from the Project Structure panel.
