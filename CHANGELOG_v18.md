# 3D Model Editor v18

## Animation tab added

- Added bottom **Animation** tab.
- Added compact timeline controls: Play/Pause, Stop, Speed, Zoom, Add Key, Auto Key, current frame.
- Auto Key is enabled by default.
- Manual Add Key captures the selected object/component object at the current frame.
- Playback works without selecting an object and applies all animated tracks together.
- Timeline supports interpolation between keyframes for:
  - Position X/Y/Z
  - Rotation X/Y/Z
  - Scale X/Y/Z
  - Mesh geometry vertex positions when compatible
  - Material color, opacity, roughness, metalness, transparency, wireframe, clearcoat, shininess, and related numeric material properties
  - Light color and intensity
- Zoom slider changes the number of visible frames in the timeline area.
