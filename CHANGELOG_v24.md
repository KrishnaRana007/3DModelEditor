# 3D Model Editor v24

## Save / Load Project

Added full project Save/Load in custom JSON and XML formats.

### Saved state
- Objects and scene hierarchy IDs
- Mesh geometry, geometry groups, UVs, normals, materials, and textures
- Lights and light settings
- Camera position/projection and OrbitControls target
- Animation timeline/keyframes
- Current mode/tool and selected object IDs
- Modeling tool settings

### UI
- File menu now contains Save Project JSON, Save Project XML, Load Project JSON, and Load Project XML.
- Ctrl+S exports JSON.
- Ctrl+O opens JSON project import.
