3D Model Editor Guide

1. Project layout

This editor is an offline browser-based 3D Model Editor built with local Three.js, pure JavaScript, HTML, and CSS. The left toolbar contains fast access tools, the center viewport displays the scene, the right Project Structure manages hierarchy, and the bottom tabs contain Inspector, Material Asset, Properties, Animation, and History.

2. Viewport navigation

  1. Alt + left mouse drag orbits the scene.
  2. Middle mouse drag pans the scene.
  3. Mouse wheel zooms in and out.
  4. Reset View restores the editor camera to the default position.

3. Selection and modes

  1. Right-click the viewport and open Mode.
  2. Choose Object, Face, Edge, or Vertex.
  3. Click to select one item. Use Shift + click to select multiple items.
  4. Drag a rectangle in the viewport to select multiple objects/components.
  5. Face, edge, and vertex modes show hover illumination before selection and selected color after click.

4. Left toolbar

  1. Select hides the gizmo and uses wireframe/component highlights.
  2. Translate shows the move gizmo for selected objects/components.
  3. Scale shows the scale gizmo.
  4. Rotate shows the rotate gizmo.
  5. Object opens a right-side flyout with primitives such as cube, sphere, plane, cylinder, cone, torus, and other shapes.
  6. Light opens light types such as Ambient, Directional, Hemisphere, Point, Spot, and Rect Area.
  7. Camera opens Perspective and Orthographic camera creation options.

5. Project Structure / Scene Hierarchy

  1. Click an item to select it.
  2. Double-click a name or use the rename icon to rename it.
  3. Use the eye icon to hide/show objects.
  4. Use the lock icon to stop selection/transforms for that object.
  5. Use duplicate and delete icons to copy or remove hierarchy items.
  6. Drag one object onto another to make it a child. Drag it onto Scene Root to unparent.

6. Modeling tools

  1. Extrude: Switch to Face/Edge/Vertex mode, select components, then Tools → Extrude. Use the Translate gizmo or Properties tab Extrude Amount X/Y/Z to control the extrusion. Enable Combine to extrude adjacent selected faces as one connected region.
  2. Bevel: Select Face/Edge/Vertex components and apply Bevel. Use Bevel Amount in Properties to soften the selected area.
  3. Chamfer: Similar to Bevel but with a stronger straight offset.
  4. Multicut: In Face mode, selected triangulated faces are split through their center into smaller triangles.
  5. Mirror: Mirror X/Y/Z duplicates selected objects in Object mode or appends mirrored component geometry in component modes.
  6. Delete: Deletes selected objects or selected geometry components. Can also be triggered by the Delete Selected button.
  7. Duplicate: Duplicates selected objects or component geometry. Shortcut: Ctrl + D.

7. Snap, Align, UV, Texture and Array tools

  1. Snap: Enable snap from Properties, set grid/rotation/scale step, then transform selected objects or components. Use Apply Snap Now for immediate snapping.
  2. Align: Select multiple objects/components, choose axis X/Y/Z and Min/Center/Max alignment in Properties, then apply.
  3. UV Mapping: Select mesh objects, choose Box, Planar, Spherical, or Cylindrical projection, then adjust Repeat, Offset, and Rotation.
  4. Texture Controls: After uploading a texture in Material Asset, use Repeat, Offset, Rotation, Wrap Mode, and Flip Y to tune the texture.
  5. Multi-material Face Assignment: Go to Face mode, select faces, open this tool, choose material type/color, and assign material only to selected faces.
  6. Array: Select an object, choose Linear or Circular mode, then set count, offset/radius/axis/angle. Linear creates repeated objects in a line; Circular arranges copies around an axis.

8. Material Asset tab

  1. Select a mesh object.
  2. Open Material Asset.
  3. Choose material type such as Basic, Lambert, Phong, Standard, Physical, Toon, Matcap, Normal, Depth, Distance, or Shadow.
  4. Edit color, opacity, roughness, metalness, transparency, shininess, side, wireframe, and other properties supported by the selected material.
  5. Use Upload Texture to apply an image from your system directly onto the selected object.

9. Animation tab

  1. Select an object and open Animation.
  2. At frame 0 click Add Key or use Auto Key.
  3. Move the seekbar to another frame and change position, rotation, scale, material color, opacity, roughness, metalness, transparency, light/camera properties, or compatible geometry data.
  4. Press Play or Spacebar to preview interpolation.
  5. Animation plays all keyed objects even when no object is selected.

10. Dope Sheet

  1. Select an animated object and open Animation.
  2. Rows appear for Position, Rotation, Scale, Material Color, and Opacity.
  3. Click a diamond key to jump to its frame.
  4. Drag a diamond to move a keyframe.
  5. Double-click or right-click a diamond to delete the keyframe.

11. File, project save/load and model import/export

  1. Save Project JSON/XML stores objects, geometry, materials, textures, lights, cameras, animation keys, selected mode/tool, hierarchy, and camera state.
  2. Load Project JSON/XML restores the saved editor state.
  3. Import OBJ / GLTF / GLB / STL adds external models into the editor scene.
  4. Export OBJ / GLTF / GLB / STL exports selected mesh objects; if nothing is selected, it exports all editable mesh objects.

12. Camera Objects

  1. Add Perspective or Orthographic cameras from the Camera flyout or context menu.
  2. Select a camera and choose View → View Through Selected Camera.
  3. Use Exit Camera View to return to the editor camera.
  4. Camera preview frame appears during camera view.
  5. Camera transforms and camera properties such as FOV, Zoom, Near, Far, and Ortho Size can be animated.

13. View Modes

  1. Solid: Simple solid display.
  2. Wireframe: Shows mesh wireframes.
  3. Material Preview: Shows actual object materials.
  4. Rendered Preview: Uses richer lighting/display preview.
  5. X-Ray: Makes objects semi-transparent for through-viewing.
  6. Normal View: Displays normal-based colors.
  7. UV View: Displays UV preview pattern; auto-generates simple UVs where needed.

14. Grid and Measurement tools

  1. Open Tools → Grid / Measurement.
  2. Set grid size, grid divisions, and unit system: meter, centimeter, millimeter, inch, or foot.
  3. Ruler/Distance: Click two points to measure length.
  4. Angle: Click three points; the second point is the angle vertex.
  5. Use Clear Measurements to remove markers, lines, and labels.

15. Undo / Redo and History panel

  1. Use Ctrl + Z to undo and Ctrl + Y or Ctrl + Shift + Z to redo.
  2. Open the bottom History tab to see commands such as Add Cube, Move Object, Extrude Face, Apply Material, Add Keyframe, Import Model, Rename, Delete, Parent, and more.
  3. Click Start State to undo all commands.
  4. Click any command row to jump the editor back/forward to that history point.
  5. Undoable commands are shown as executed; redoable commands are shown as undone.
