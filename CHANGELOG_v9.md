# 3D Model Editor v9 - Flyout Create Menus and Styled Scrollbars

## Added / Changed

- Changed the left **Object** button behavior from an inline/downward dropdown to a context-style flyout that opens on the right side of the left Tools panel.
- Changed the left **Light** button behavior from an inline/downward dropdown to the same right-side flyout style.
- Added safer flyout positioning so menus stay inside the browser window when the editor is resized.
- Added custom modern scrollbars across the editor using WebKit scrollbar styling and Firefox `scrollbar-color` fallback.

## Preserved

- Fully offline Three.js setup.
- Default cube at origin.
- Alt + mouse orbit.
- Rectangle select without Ctrl.
- Select tool hides gizmo.
- Object, Face, Edge and Vertex modes.
- Hover/selected component highlighting.
- Basic modeling tools and Guide modal.
- Command-pattern Undo/Redo.
